# 高级网络配置

[[toc]]

## 配置概览

与网络相关的配置如下：

- proxy：代理服务器配置
- httpAgent：Node.js 底层连接控制
- maxRedirects：重定向次数限制
  ::: code-group

```js [通用配置]
// 【代理】代理服务器配置
proxy: {
  protocol: 'https',
  host: '127.0.0.1',
  port: 9000,
  auth: {  // 代理认证信息
    username: 'mikeymike',
    password: 'rapunz3l'
  }
},

```

```js [Node专用配置]
// 【Node专用】自定义HTTP/HTTPS代理
httpAgent: new http.Agent({ keepAlive: true }),
httpsAgent: new https.Agent({ keepAlive: true }),
// 【Node专用】最大重定向次数（0表示不跟随重定向）
maxRedirects: 5,
```

:::

## 代理服务器配置-proxy

配置类型如下：

```js
export interface AxiosProxyConfig {
  host: string;
  port: number;
  auth?: AxiosBasicCredentials;
  protocol?: string;
}
```
