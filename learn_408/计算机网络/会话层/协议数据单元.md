# 会话层-协议数据单元

[[toc]]

会话层的 PDU 称为 ​SPDU（Session Protocol Data Unit）​，包含以下字段：

| 字段       | 作用                                           |
| ---------- | ---------------------------------------------- |
| 会话 ID    | 唯一标识一个会话（如 SSH 会话 ID）。           |
| 控制信息   | 管理会话状态（开始、结束、同步请求）。         |
| 检查点标识 | 标记数据同步点（如文件传输中的断点续传位置）。 |
| 用户数据   | 承载应用层数据（如加密的 RPC 调用参数）。      |

```bash
| 会话ID (4B) | 控制标志 (1B) | 检查点编号 (2B) | 数据长度 (2B) | 用户数据 (变长) |
```
